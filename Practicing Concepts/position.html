This is a static positioned element. It is default positioning system for all elements.
This is relative positioned element. A relative positioned element is one which is relative to its normal position. 100px from top & 100px from left.
This is fixed positioned element. A fixed positioned element is one in which the element is positioned relative to the viewport. Which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.

This is absolute positioned element. A absolute positioned element is one in which the element is positioned relative to the nearest positioned ancestor.
